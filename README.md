# Robotics Learning Platform



### Project Description

This project is a web platform developed with Laravel that allows a robotics school to manage courses, students, teachers and robotics kits.

The platform organizes students into groups and assigns robotics courses that can be consulted directly on the platform.





### ER Diagram





Entities included in the system:



* Users
* Groups
* Courses
* Robotics Kits



Relationships:



* A user belongs to a group
* A group has many courses
* A course has a robotics kit
